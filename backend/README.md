# SunoHub Backend

This is the Node.js + Express backend for SunoHub.

## Dependencies

- `express`
- `mongoose`
- `cors`
- `dotenv`
- `jsonwebtoken`
- `multer`
- `bcryptjs`
- `socket.io`

## Development Dependencies

- `nodemon`

## Startup Commands

To run the server in a development environment, use the following command:

```bash
nodemon server.js
```
